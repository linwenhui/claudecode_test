SLIDE 03 · HUI HTML DECK

信息架构

三层拆解：结构、视觉、交互

结构层

slide 分节 + 侧栏状态同步。

视觉层

玻璃、渐变大标题、暗色代码区对比。

交互层

目录与箭头切换单屏，长内容在屏内滚动。

if ok:
    run()
else:
    fix()
for x in items:
    step(x)
while n < 3:
    n += 1

How to explain

用「先判断再分支、需要重复就用循环」概括控制流，听众最容易跟上。
