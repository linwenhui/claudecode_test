# Remotion Best Practices

Domain knowledge for building videos with Remotion + React.

## When to Use

- Build or refactor Remotion projects
- Create composition structures and animation timelines
- Handle audio, subtitles, trimming, transitions, and rendering
- Improve reliability and maintainability of Remotion code

## Quick Start

```bash
npx create-video@latest --yes --blank --no-tailwind my-video
npx remotion studio
```

Optional one-frame sanity check:

```bash
npx remotion still [composition-id] --scale=0.25 --frame=30
```

## Key Areas

- Composition architecture and metadata
- Animation timing and transitions
- Audio/video assets and trimming
- Captions, voiceover, and effects
- Rendering/export workflow
